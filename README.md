# Port PEAR package [HTMLValidator](http://pear.php.net/package/Services_W3C_HTMLValidator)


### Requirements:

- PHP >= 5.3

### Installation:

- Add to composer.json:

```json
{
    "require": {
        "gemorroj/htmlvalidator": "dev-master"
    }
}
```
- Install project:

```bash
$ php composer.phar update gemorroj/htmlvalidator
```
